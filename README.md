# CS247: Data Mining
该项目是CS247数据挖掘课程(2022秋)的大作业. 项目要求使用多种模型进行自然语言的情绪分析, 本项目中尝试了Bayes、SVM、LSTM、BiLSTM、BiLSTM-Attention、RNN、BERT等模型, 并从数据预处理优化、特征压缩的对比与优化、模型的调参与网络分析、预测结果分析四个方面的深入分析与研究.
